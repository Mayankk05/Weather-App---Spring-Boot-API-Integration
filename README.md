# Weather-App---Spring-Boot-API-Integration

A simple full-stack weather application that fetches real-time weather data using a third-party Weather API. Built with Spring Boot for the backend and HTML/CSS/JavaScript for the frontend.

Tech Stack:
Backend: Spring Boot, REST API, Spring Web

Frontend: HTML, CSS, JavaScript

API: OpenWeatherMap (or any other public weather API)
Features:
Search weather by city name

Real-time temperature, humidity, and condition display

Clean UI with responsive design
